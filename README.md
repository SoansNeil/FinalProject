# FinalProject

## Description
<ul>This application allows users to interact with a map of the world. The map implements interactive points which represent soccer teams in the world. A user is able to interact with the points or search for the team using a searchbar which will be followed by a quick overview of the history/achievements of the team.</ul>

## Tech Stack
<li>Frontend: React.js</li>
<li>Backend: Node.js + Express.js</li>
<li>Database: PostgreSQL + PostGIS</li>
<li>Deployment: Google Cloud (GCP)</li>
